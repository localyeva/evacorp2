== Description ==

= Export DB to a new domain =

With Export DB to a new domain you can move your websites faster and save time.
Edit and export your database in 1 step.

The current domain database stayed untouched its don't change your current domain database in anyway.

Also its let you to backup the current domain database.

= More Tools =
* Search in entire database
* Search in database table


== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Export DB to a new domain" page under Tools in the admin menu
